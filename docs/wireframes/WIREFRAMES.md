## Component Library Reference

These wireframes assume a component library with:

- **Navigation**: Sidebar (desktop), Bottom tabs (mobile)
- **Cards**: Candidate cards, Job cards, Stat cards
- **Forms**: Input fields, Select dropdowns, File upload
- **Modals**: Confirmation, Forms, Alerts
- **Tables**: Sortable, Filterable, Paginated
- **Buttons**: Primary, Secondary, Destructive, Icon
- **Status indicators**: Badges, Pills, Progress bars
